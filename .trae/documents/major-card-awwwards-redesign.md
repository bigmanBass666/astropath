# 专业卡片 Awwwards 级美学重设计计划

## 创意概念：Data Sculpture（数据雕塑）

> 专业卡片应像一件精密雕刻的数据雕塑——每个元素都有目的和重量，信息通过层次而非堆砌来呈现。视觉锚点是**专业名称本身**，数据区域是支撑它的底座。

---

## 当前问题诊断（对比院校卡片）

| 问题 | 院校卡片 | 专业卡片（当前） |
|------|---------|-----------------|
| 头部区域 | 名称 + 排名徽章，干净利落 | Checkbox + 类别标签，工具感太强 |
| 名称区域 | 单一名称，视觉聚焦 | 中英双语但层次不够分明 |
| 元信息行 | 国家位置（有图标） | 学位·学制（与数据区重复） |
| 数据区域 | TUITION \| RATE 双列，视觉统一 | DURATION \| SALARY 双列，OK 但缺乏冲击力 |
| 底部操作 | 实心按钮 "+ 收藏"，有存在感 | 纯文本链接 "查看详情 →"，太弱 |
| 悬停效果 | 3D 倾斜 + 阴影 + accent 条加高 | 同机制但整体感觉平淡 |
| 配色合规 | ✅ Slate 徽章实底白字 | ❌ 类别标签用彩色背景 |

---

## 重设计方案（6 大改造模块）

### 模块 1：头部区域重构 — 去工具化

**现状问题：** Checkbox 直接放在头部，看起来像表单而非精品卡片。

**改造方案：**
- 将 `el-checkbox` 从头部移出，改为卡片左上角的**悬浮圆形对比按钮**
- 类别标签改为**描边风格**（outline），不再使用彩色填充背景
- 新增一个**序号/ID 视觉锚点**在右上角（如 `#01`、`#02`），用 mono 字体

```
改造前: [☑ 对比] [类别标签]          ← 工具感强
改造后: [◉]        [#03] [ENG]      ← 精品感，◉ 是悬浮对比按钮
```

**涉及修改：**
- Template: `ud-major__head` 区域重构（L415-423）
- Styles: `.ud-major__head`, `.ud-major__check`, `.ud-major__cat` 重写

---

### 模块 2：名称区域升级 — 视觉冲击力

**现状问题：** 中英文名称排在一起但缺乏层次感。

**改造方案：**
- 中文名称：字号从 `17px` 提升到 `19px`，字重保持 `700`
- 英文名称：缩小到 `10px`，改用斜体 + opacity 降低，加 `/` 前缀
- 在名称下方添加一条**微妙的 accent 装饰线**（`::after` 伪元素）
- 整体增加 letter-spacing 的负值，让字更紧凑有力

```
改造前: 计算机科学  Computer Science     ← 平铺直叙
改造后: 计算机科学                    ← 主标题突出
         / Computer Science           ← 副标题优雅退让
         ─────────────────────         ← accent 装饰线
```

**涉及修改：**
- Template: `ud-major__name` 区域微调（L425-430）
- Styles: `.ud-major__name`, `.ud-major__name-cn`, `.ud-major__name-en` 重写

---

### 模块 3：元信息行精简 — 去冗余

**现状问题：** `学位类型 · 学制` 这行信息中学制与下方数据区 DURATION 重复。

**改造方案：**
- 移除元信息行中的「学制」部分，只保留**学位类型**
- 改为更紧凑的**胶囊式展示**：用一个带背景的小 badge 展示学位
- 字号缩小到 `11px`，用 mono 字体

```
改造前: ● 本科 · 本科4年/硕士2年          ← 冗余
改造后: [ BACHELOR ]                     ← 精炼
```

**涉及修改：**
- Template: `ud-major__meta` 区域简化（L431-433）
- Styles: `.ud-major__meta`, `.ud-major__dot` 重写

---

### 模块 4：数据区域增强 — 信息密度与美感并存

**现状问题：** 功能完备但视觉上不够"高级"。

**改造方案：**
- 保持双列布局（DURATION \| SALARY）
- 给数据区添加**左侧 3px accent 色竖条**作为视觉引导
- 数值字号从 `14px` 提升到 `16px`，更醒目
- SALARY 值添加**渐变色效果**（Amber 渐变，符合配色系统）
- Label 保持 `9px` uppercase mono 不变
- 背景改为极微妙的渐变（`#FAFBFC → #F8FAFC`）

```
改造前: ┌─────────────────────────┐
         │ DURATION    SALARY     │
         │ 4年/2年    ¥20K-50K/月 │
         └─────────────────────────┘

改造后: ┃┌─────────────────────────┐
         ││ DURATION    SALARY     │
         ││ 4年/2年    ¥20K-50K/月 │  ← 左侧 accent 竖条
         └└─────────────────────────┘
```

**涉及修改：**
- Template: `ud-major__data` 区域（L434-443）— 可能需要加装饰元素
- Styles: `.ud-major__data` 及内部样式全面重写

---

### 模块 5：底部操作区重设计 — 从弱链接到强按钮

**现状问题：** "查看详情 →" 文本链接太弱，没有行动召唤力。

**改造方案：**
- 将文本链接替换为**实心按钮**，风格对齐院校卡片的 `ud-card__action`
- 按钮文案改为 `DETAILS`（英文 mono 字体，更国际化）
- hover 时按钮右移 + 阴影出现（Back Out 缓动）
- 按钮左侧留空，右侧对齐（右对齐更有设计感）

```
改造前: ───────────────────── 查看详情 →    ← 弱
改造后: ──────────────────── [ DETAILS → ]  ← 强
```

**涉及修改：**
- Template: `ud-major__foot` 区域（L445-450）
- Styles: `.ud-major__foot`, `.ud-major__link` → 改为 `.ud-major__action`

---

### 模块 6：悬停效果与微交互升级

**改造方案：**

**a) 入场动画（可选，如果页面支持）：**
- 卡片交错淡入 + 轻微上移（staggered reveal）

**b) 悬停增强：**
- 3D 倾斜保持不变（已有）
- accent 条高度从 `3px→4px` 改为 `3px→6px`（更明显）
- 添加**微妙的 border glow**（box-shadow 加一层 Amber 低透明度）
- 名称区域的装饰线在 hover 时**从左向右展开**（CSS transition width）

**c) 对比按钮交互：**
- 默认状态：空心圆 + 细边框
- 选中状态：填充 Amber 色 + 白色对勾
- hover 时轻微放大（scale 1.1）

**涉及修改：**
- Styles: `.ud-major:hover`, `.ud-major__accent`, 新增装饰线动画

## 配色合规检查

所有新增颜色必须来自现有变量：

| 用途 | 变量 | 色值 |
|------|------|------|
| 主文字 | `--color-text-primary` | #0F172A |
| 次文字 | `--color-text-secondary` | #475569 |
| 弱文字 | `--color-text-tertiary` | #94A3B8 |
| 强调色 | `--color-accent` | #D97706 |
| 边框 | `--color-border-light` | #F1F5F9 |
| 背景 | `--color-surface` | #FFFFFF |
| 次背景 | `--color-background-alt` | #F8FAFC |
| 类别标签 | 改为 outline + `--color-solid` | #0F172A |

**❌ 禁止使用的颜色：** 任何不在 variables.css 中的新颜色
