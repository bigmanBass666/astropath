# 院校数据添加指南

## 任务概述
按照QS世界大学排名，为留学申请系统添加新的院校数据，确保数据真实、准确、可溯源。

## 数据标准

### 1. 必填字段
每个院校必须包含以下字段：

```javascript
{
  id: Number,                    // 唯一ID，递增
  name: String,                  // 院校中文名称
  country: String,               // 国家/地区
  major: String,                 // 热门专业
  ranking: String,               // QS排名，格式: 'QS #数字'
  match: Number,                 // 匹配度 0-100
  deadline: String,              // 申请截止日期 YYYY-MM-DD
  tuition: String,               // 学费（含货币符号）
  acceptanceRate: String,        // 录取率，格式: '数字%'
  requirements: [String],        // 申请要求数组，4条左右
  website: String,               // 官网URL
  description: String,           // 学校简介 50-100字
  type: String,                  // 学校类型：'公立大学'/'私立大学'
  students: String,              // 学生人数，格式: '约数字人'
  image: String,                 // 图片URL（可留空）
  sources: {                     // 信息来源（必须可溯源）
    ranking: { url: String, label: String },
    tuition: { url: String, label: String },
    acceptance: { url: String, label: String },
    requirements: { url: String, label: String }
  }
}
```

### 2. 数据验证规则

#### 排名 (ranking)
- 格式必须为 `QS #数字`
- 来源：QS官网最新排名
- 示例：`'QS #1'`, `'QS #15'`, `'QS #100'`

#### 学费 (tuition)
- 必须包含货币符号
- 使用院校官网或权威教育网站数据
- 货币符号规范：
  - 美元: `$57,590`
  - 英镑: `£35,000`
  - 瑞士法郎: `CHF 1,500`
  - 新加坡元: `SGD 32,850`
  - 澳元: `AUD 42,000`
  - 加元: `CAD 58,160`
  - 港币: `HKD 171,000`
  - 人民币: `¥30,000`

#### 录取率 (acceptanceRate)
- 格式必须为 `数字%`
- 来源：院校官网、College Tuition Compare、UnivStats等
- 示例：`'3.6%'`, `'15%'`, `'43%'`
- 注意区分本科/研究生录取率

#### 申请要求 (requirements)
- 数组格式，包含4条左右
- 必须包含：
  1. GPA要求（注明是最低要求还是录取者平均）
  2. 语言要求（TOEFL/IELTS，注明单项分数）
  3. 其他学术要求
  4. 申请材料要求
- 示例：
  ```javascript
  requirements: [
    'GPA 3.8+ (录取者平均)',
    'TOEFL 100+ / IELTS 7.0+',
    '科研经历优先',
    '推荐信3封'
  ]
  ```

#### 信息来源 (sources)
- 每个数据点必须有可访问的来源URL
- 来源标签要简洁明了
- 优先级：
  1. 院校官网（最权威）
  2. QS官网（排名）
  3. 权威教育网站（College Tuition Compare, Collegedunia等）
  4. 留学机构网站（新东方前途出国等）

## 添加流程

### 步骤1: 信息收集
1. 确认QS排名
   - 访问 https://www.qs.com/rankings/
   - 记录最新排名

2. 收集院校信息
   - 学费：院校官网 -> Admissions -> Tuition & Fees
   - 录取率：院校官网或 College Tuition Compare
   - 申请要求：院校官网 -> Admissions -> Requirements
   - 学生人数：院校官网 About 页面

3. 记录来源URL
   - 每个数据点记录来源网址
   - 确保链接可访问

### 步骤2: 数据格式化
按照数据标准格式化收集的信息：

```javascript
{
  id: 20,  // 递增ID
  name: '某某大学',
  country: '国家',
  major: 'Computer Science',
  ranking: 'QS #21',
  match: 75,  // 根据排名估算
  deadline: '2025-01-15',
  tuition: '$50,000',
  acceptanceRate: '20%',
  requirements: [
    'GPA 3.5+ (最低要求)',
    'TOEFL 90+ / IELTS 7.0+',
    '相关专业背景',
    '个人陈述'
  ],
  website: 'https://example.edu',
  description: '学校简介，50-100字。',
  type: '公立大学',
  students: '约30,000人',
  image: '',
  sources: {
    ranking: { 
      url: 'https://www.qs.com/rankings/', 
      label: 'QS 2025排名' 
    },
    tuition: { 
      url: 'https://example.edu/tuition', 
      label: '某某大学官网' 
    },
    acceptance: { 
      url: 'https://collegedunia.com/...', 
      label: 'Collegedunia' 
    },
    requirements: { 
      url: 'https://example.edu/admissions', 
      label: '某某大学招生要求' 
    }
  }
}
```

### 步骤3: 更新代码
1. 打开 `src/data/schools.js`
2. 在 `schoolsData` 数组末尾添加新院校数据
3. 确保 `id` 唯一且递增

### 步骤4: 验证
1. 检查数据格式是否正确
2. 验证所有URL可访问
3. 确认数据真实性
4. 测试前端显示效果

## 匹配度估算规则

根据QS排名估算 match 值：

| 排名范围 | match值 |
|---------|---------|
| Top 10  | 85-95   |
| 11-20   | 75-85   |
| 21-50   | 65-75   |
| 51-100  | 55-65   |
| 100+    | 45-55   |

## 常见问题

### Q1: 如何处理研究生和本科生费不同的情况？
A: 优先使用研究生学费，因为系统主要面向研究生申请。在学费字段中可以注明范围，如 `$50,000-$60,000`。

### Q2: 录取率数据找不到怎么办？
A: 依次尝试以下来源：
1. 院校官网 Common Data Set
2. College Tuition Compare
3. Collegedunia
4. 留学论坛和机构网站
如果确实找不到，可以使用估算值，但必须在来源中注明"估算"。

### Q3: 申请要求如何简化？
A: 提取最重要的4条要求：
1. GPA（必须）
2. 语言成绩（必须）
3. 标准化考试（如GRE/GMAT，可选）
4. 其他重要要求（科研经历、推荐信等）

### Q4: 如何处理多个校区？
A: 主校区单独作为一个院校，分校区可以单独添加，在名称中注明，如"某某大学（某某校区）"。

## 示例：添加新院校

假设要添加QS排名第21的大学：

```javascript
// 1. 收集信息
// QS排名: #21
// 学费: $55,000 (官网)
// 录取率: 25% (College Tuition Compare)
// 申请要求: GPA 3.5+, TOEFL 90+, 个人陈述, 推荐信2封

// 2. 格式化数据
const newSchool = {
  id: 20,
  name: '某某大学',
  country: '美国',
  major: 'Engineering',
  ranking: 'QS #21',
  match: 75,
  deadline: '2025-01-15',
  tuition: '$55,000',
  acceptanceRate: '25%',
  requirements: [
    'GPA 3.5+ (最低要求)',
    'TOEFL 90+ / IELTS 7.0+',
    '个人陈述',
    '推荐信2封'
  ],
  website: 'https://example.edu',
  description: '某某大学是美国顶尖研究型大学，在工程和计算机领域享有盛誉。',
  type: '公立大学',
  students: '约35,000人',
  image: '',
  sources: {
    ranking: { 
      url: 'https://www.qs.com/rankings/', 
      label: 'QS 2025排名' 
    },
    tuition: { 
      url: 'https://example.edu/tuition', 
      label: '某某大学官网' 
    },
    acceptance: { 
      url: 'https://www.collegetuitioncompare.com/...', 
      label: 'College Tuition Compare' 
    },
    requirements: { 
      url: 'https://example.edu/admissions', 
      label: '某某大学招生要求' 
    }
  }
}

// 3. 添加到 schoolsData 数组
schoolsData.push(newSchool)
```
