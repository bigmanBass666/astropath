# Writer's Studio — Materials 页面重设计方案

> 日期: 2026-04-07 | 状态: ✅ 已确认，实施中
> 核心目标: 将 Materials 页面从「功能堆砌+外壳」改造为「浑然天成的沉浸式写作工作室」

## 设计决策记录

| 决策点 | 选择 | 理由 |
|--------|------|------|
| 核心体验 | 文书创作为主 | 创新分核心（AI交互30分） |
| 交互形态 | 全屏编辑器 + 侧边抽屉 | 类似 Notion/Cursor 的沉浸体验 |
| AI角色 | 写作伙伴常驻面板 | 类似 GitHub Copilot Chat |
| Modal策略 | 零Modal | 所有操作内联或抽屉式 |

## 架构设计

### 组件拆分

```
Materials.vue (主容器 ~250行)
├── WelcomeWizard.vue      — 首次进入向导（文书类型→模板选择）
├── StudioHeader.vue       — 顶栏（文档名/模板/保存/导出/材料入口）
├── EditorPane.vue         — 全屏编辑区 + 格式工具栏
├── AICompanionPanel.vue   — AI 写作伙伴面板（右侧常驻/可折叠）
├── MaterialsDrawer.vue    — 材料清单 + 申请进度（右侧滑出抽屉）
└── StudioStatusBar.vue    — 底部状态栏（字数/保存/预览/版本）

useMaterialsState.ts       — 所有状态逻辑抽取 (~400行)
```

## 各组件详细设计

### 1. StudioHeader
- 高度48px, sticky top, 毛玻璃背景
- 左侧: 菜单按钮 + 文档标题(可编辑, mono字体) + 模板名
- 右侧: 保存 + 导出PDF + 材料徽章 + AI切换

### 2. EditorPane
- 全屏contenteditable, 820px最大宽度居中, 80x60px内边距
- 浮动工具栏(icon-only) + Slash Command支持
- 编辑/预览内联切换(非dialog)

### 3. AICompanionPanel
- 默认360px宽, 可拖拽调整, 可折叠
- 快捷指令芯片组 + 流式对话 + 建议采纳
- 上下文感知(自动读取选中文字)

### 4. MaterialsDrawer
- 400px宽, 右侧滑入, Expo Out动画
- 上半: 材料清单手风琴, 下半: 申请进度时间线
- 点击"PS"→关闭抽屉→聚焦编辑器

### 5. StudioStatusBar
- 32px固定底部, 毛玻璃背景
- 字数(mono) + 阅读时间 + 保存状态 + 预览/版本

### 6. WelcomeWizard
- 首次进入全屏覆盖层
- Step1: 选文书类型(4卡片) → Step2: 选模板 → 进入编辑器

## 视觉规范
- 配色: Slate #0F172A + Amber #D97706, 零新色
- 字体数据: JetBrains Mono, 标题: Playfair Display
- 动效入场: cubic-bezier(0.16,1,0.3,1), 微交互: Back Out
- 噪点: fractalNoise opacity 0.025

## 创新亮点
1. 全屏零干扰写作环境
2. AI写作伴侣常驻面板
3. Slash Command (/命令菜单)
4. 零Modal设计
5. 材料-编辑联动
6. 自动保存+实时状态

## 响应式
- ≥1200px: 编辑器+右侧AI面板(360px)
- 1024-1199px: 编辑器+AI面板(280px可折叠)
- 768-1023px: 全屏编辑器, AI变底部抽屉
- ≤767px: 全屏编辑器, AI和材料均底部全宽抽屉
