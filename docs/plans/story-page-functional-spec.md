一个纪念性页面，用于记录和呈现智途 AstroPath 项目从构思到比赛完成的完整历程。以第三人称视角叙述，由AI书写，但传达开发者真实的情感投入与付出。

## 3. 内容模块规划

### 模块A：项目概览
- **项目全称**: 智途 AstroPath
- **项目副标题**: 一站式智能留学规划平台
- **开发周期**: 从首次commit到比赛当天的完整时间跨度
- **比赛信息**: 人工智能学院第四届科创节网页设计大赛（2026年4月7日）
- **一句话总结**: 本项目诞生的初衷与最终达成的目标

### 模块B：开发历程时间线
按时间顺序展示关键节点：

1. **项目启动** - 最初的想法与动机
2. **技术选型** - 为什么选择Vue 3 + Vite + Element Plus
3. **核心功能开发** - 背景评估、AI选校、时间规划、材料中心、AI对话
4. **设计迭代** - 从初稿到Awwwards级设计标准的演进
5. **比赛准备** - 答辩稿、演示流程、最终优化
6. **比赛日** - 上台展示、等待结果

每个节点包含：日期、事件描述、当时的挑战与突破

### 模块C：量化的心血（开发数据）
自动统计并展示：

- **代码统计**
  - 总代码行数（JS/Vue/TS/CSS）
  - 文件数量
  - 组件数量
  
- **Git提交数据**
  - 总提交次数
  - 开发天数跨度
  - 最活跃的开发日期
  
- **内容数据**
  - 院校数据条数
  - 专业数据条数
  - AI对话预设角色数

### 模块D：技术亮点
列出开发过程中值得骄傲的技术决策：

- AI流式响应架构（SSE解析、思考模式、队列管理）
- 多Provider AI配置系统
- 状态管理的composable设计模式
- 氛围分级设计系统
- 响应式布局与移动端适配

### 模块E：设计演进（可选）
如果有截图存档：
- 首页设计的迭代版本对比
- 关键页面的初稿vs终稿

### 模块F：致谢与感言
- **致谢对象**: 开源社区、设计灵感来源、技术文档
- **开发者感言**: 以第三人称转述开发者的真实感受
  - 开发过程中的辛苦与坚持
  - 对项目投入的情感
  - 比赛后的心情
  - 这不是结束，而是一次完整的自我证明

## 4. 交互功能

### 4.1 时间线交互
- 支持横向或纵向滚动浏览
- 每个节点可点击展开详情
- 当前节点高亮显示

### 4.2 数据可视化
- 开发数据以数字动画形式呈现（从0增长到实际值）
- 代码统计可用简单的条形图或数字卡片展示

### 4.3 平滑滚动
- 页面内各模块之间支持平滑滚动跳转
- 滚动时触发元素入场动画（由视觉设计决定具体效果）

### 4.4 返回入口
- 页面底部提供返回首页的链接

## 7. 性能考虑

- 页面为静态展示性质，无需频繁数据更新
- 图片资源（如有设计演进截图）需要懒加载
- 数字动画在元素进入视口时触发，避免初始加载压力

---

## 8. 无障碍要求

- 所有内容可被屏幕阅读器识别
- 时间线节点需要有明确的焦点状态
- 支持键盘导航
- 尊重用户的`prefers-reduced-motion`设置
