# 人工智能学院第四届科创节网页设计大赛广州（从化）校区

## 第四届科创节 网页设计大赛

**人工智能学院 博雅学院**

### 活动目的

为响应国家就业优先战略与教育数字化发展核心导向，以网页设计实践为载体，搭建数字化技能与升学规划、就业发展的高效衔接平台，人工智能学院将联合博雅学院开展网页设计大赛。本活动立足大学生生涯发展核心需求，通过主题化设计实践，引导学生精准把握国家战略对高素质人才的能力要求，深度理解升学政策导向、就业市场趋势与数字化人才供需逻辑。切实呼应国家“培养数字化、复合型人才”的发展目标。

## Part. 2 比赛作品主题

**网页设计作品选题（以下三选一）**

1. **打造出国深造一站式资讯与国际留学规划服务类网页**，为学子提供海外深造全流程指引。
2. **搭建考研备考资源整合与院校数据决策类网页**，开展专业化指引与实践服务，助力学生高效规划国内升学路径。
3. **设计应届生求职赋能与行业就业洞察类网页**，为毕业生搭建直通职场的综合服务平台。

---

## Part. 3 作品要求

1. **参赛要求**：参赛队伍需设计一款兼具创新性与实用性的网页，在规定时间内完成制作与展示。
2. **参赛选手自定一个站点名称**，静态网页格式为html，动态网页格式可以为asp、jsp、php等。要求主题鲜明突出，创意新颖，页面结构美观。
3. **学生组队参赛**（可跨学院、跨专业、年级自由组队）。每队队员2-3人。
4. **作品提交内容**：源代码；作品介绍书word。
5. **设计要求**：
   - （1）作品内容积极向上，页面风格统一，交互性强，整体布局美观大方，符合本次比赛作品征集要求。
   - （2）制作软件不限；通过合理运用网页设计的各种技术体现良好的艺术性和实用性。
   - （3）页面深度尽量在3层左右，全部文件的总容量尽量在100M左右。
   - （4）要求能在win7或win10等操作系统上运行。
   - （5）所有作品必须原创，严禁抄袭，模仿，拷贝他人的作品，一经查出，取消其参赛资格。
   - （6）逾期未提交则视为放弃比赛资格。

## Part. 5 评分细则

### 网页设计（40分）
界面美观，色彩搭配协调，布局合理，符合用户操作习惯，整体风格统一。

### 功能实现（30分）
功能完整，涵盖所设定的各项功能；交互流畅，无卡顿、延迟现象，操作便捷；不存在明显错误，确保用户正常使用。

### 创新性（30分）
设计理念新颖独特，打破传统设计模式；积极应用前沿技术，如人工智能交互技术等，提升网页的独特性与竞争力。

### 展示效果（10分）
表达清晰流畅，逻辑连贯，能够准确传达作品的核心价值与亮点，让观众和评委快速理解作品内涵。
